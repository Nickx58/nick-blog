# Welcome! 👋

This is my very first blog post. I'm excited to share my thoughts on technology, programming, and the craft of building software.

## Why I Started This Blog

I've been meaning to start writing for a while now. There's something powerful about putting thoughts into words — it forces clarity and creates a record of lessons learned.

## The Tech Stack

This blog is built with:

- **Next.js 15** with the App Router for static site generation
- **Markdown** files for content (no CMS needed!)
- **Vanilla CSS** with CSS custom properties for theming
- **Vercel** for deployment

### Adding New Posts

The best part? Adding a new post is as simple as:

1. Create a new `.md` file in `content/posts/`
2. Add frontmatter (title, date, description, tags)
3. Write in Markdown
4. Push to Git — Vercel auto-deploys

```markdown
---
title: "My New Post"
date: "2026-04-08"
description: "A short description"
tags: ["example"]
---

Your content here...
```
